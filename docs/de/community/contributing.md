# Mitwirken und Lizenz

Beiträge machen die Open-Source-Community zu dem großartigen Ort, an dem gemeinsam gelernt, inspiriert und gebaut wird. Jeder Beitrag ist **sehr willkommen**.

Wenn du eine Idee hast, die das Projekt verbessert, öffne bitte ein Issue mit dem Label `enhancement`, forke das Repository und stelle einen Pull Request. Du kannst auch einfach ein Issue zur Diskussion aufmachen.

## So beitragen

1. Öffne ein Issue mit dem Label `enhancement`.
2. Forke das Projekt.
3. Lege deinen Feature-Branch an:

   ```bash
   git checkout -b feature/AmazingFeature
   ```

4. Committe deine Änderungen:

   ```bash
   git commit -m 'Add some AmazingFeature'
   ```

5. Pushe den Branch:

   ```bash
   git push origin feature/AmazingFeature
   ```

6. Öffne einen Pull Request.

Weitere Details findest du in der [CODE_OF_CONDUCT.md](https://github.com/it-at-m/agile-test-automation-framework/blob/main/CODE_OF_CONDUCT.md).
